## Git repository 이름 및 url 변경

1. git repository의 Setting 메뉴에서 Repository name 변경

2. git CMD에서 해당 프로젝트 로컬 경로로 이동 후,

   ```
   $ git remote -v
   ```

   명령어를 입력하면, 현재 프로젝트 저장소 별칭 및 url을 출력
   
3. 저장소 url 변경

   ```
   $ git remote set-url [NewRepositoryURL]
   ```
